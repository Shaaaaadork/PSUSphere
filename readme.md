# Student Organizational Management System

## :pushpin: Project Description
The student organizational management is a django based web application designed to manage colleges, students, programs, organizations, student membership within the Palawan State University.

This project helps the administrators to easily organize students groups, assign members and maintain updated records.

## Features
- Manage Colleges (add, update, search, filter)
- Manage Academic Programs under each College
- Manage Student Organizations with descriptions and college affiliation
- Register Students with program information
- Assign Students as Members of Organizations
- Admin dashboard with search and filtering tools
- Automatic generation of sample data using Faker

## :rocket: Technologies Used
- Python
- Django Framework
- SQLite / MySQL Database
- Faker Library for test data generation
